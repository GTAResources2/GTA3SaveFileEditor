GTA III Save File Editor - changes in version 0.2

(not final)

Editor
------
* Support for Android and iOS saves.
* Support for unsigned input
* Better input validation
* Internal data handling completely rewritten. Much safer.
* Better logging.
* Added command line option --log-level=<level>
* Minor visual changes
* Added recent files list
* Added support for Mac OS X
* Overall stability has been improved

General
-------
* Weather interpolation value is now adjusted via a slider
* Support for iOS and Android saves
* (mobile only) GXT strings can be selected and used as the save name

Player
------
* Improved max wanted level setting
* Wanted level can now be disabled entirely

Garages
-------
* Car colors are easier to select
